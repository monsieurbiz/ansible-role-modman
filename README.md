# Ansible Modman Role

It just installs modman.

## Requirements

* Ansible version 2.*

# Role Variables

You can override all the variables in `defaults/main.yml` in tour own vars.

## Example Playbook

    - hosts: servers
      roles:
         - { role: monsieurbiz.modman }
